# 3D Agent Office — Pilot

Pilot repo used to test source control and GitHub Pages deployment for the
[3D-Agent-Office](https://github.com/eriic-builds/3D-Agent-Office) project (a Gemini-free fork of
[The Delegation](https://github.com/arturitu/the-delegation)).

- **Repo:** https://github.com/eriic-builds/3d-agent-office-pilot
- **Live GitHub Pages site:** https://eriic-builds.github.io/3d-agent-office-pilot/

## What's here

A production build (`ui-glass-redesign` branch) of the 3D-Agent-Office app, deployed as a static
site via GitHub Pages. Uses a bring-your-own-key model client-side:

- GitHub Models (free) for agent reasoning
- Pollinations.ai (free, keyless) for image generation

Your GitHub PAT is entered in-browser and stored locally — it is never sent anywhere except
GitHub Models' API.
